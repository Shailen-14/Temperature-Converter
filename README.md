# 🌡️ Temperature Converter

A simple and interactive **Temperature Converter** built using **HTML, CSS, and JavaScript**.  
This tool allows users to quickly convert temperatures between **Celsius and Fahrenheit**, demonstrating front-end development skills, form handling, and DOM manipulation.

---

## 🚀 Features
- Convert temperatures **Celsius ↔ Fahrenheit** instantly  
- User-friendly input fields and radio button selection  
- Clear display of converted temperature  
- Simple, responsive, and clean UI  
- Lightweight and fast  

---

## 🛠️ Tech Stack
- **HTML** – Structure of the converter  
- **CSS** – Styling and responsive layout  
- **JavaScript** – Conversion logic and interactivity

## ▶️ How to Run
1. Clone the repository:
```bash
git clone https://github.com/Shailen-14/Temperature-Converter.git
```
2. Navigate to the project folder
3. Open index.html in your browser
4. Enter a temperature, select the conversion type (Celsius → Fahrenheit or Fahrenheit → Celsius), and click Submit to see the result

---

## 🧠 How It Works
1. User enters a temperature value
2. Selects the conversion type via radio buttons
3. JavaScript performs the conversion:
    - Celsius → Fahrenheit: (C × 9/5) + 32
    - Fahrenheit → Celsius: (F − 32) × 5/9
4. The result is displayed instantly below the Submit button
